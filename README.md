# [Prodigy : AI Powered Interview Mastery](https://prodigy-zeta.vercel.app/)

### Start The Front-End Server

- Clone the repo :

```
git clone https://github.com/prateekshuklaps0/AI-Powered-Interview-Mastery.git
```

- Install Dependencies :

```
cd Front-End
```

```
npm install
```

- Start the Front-End Server :

```
npm run dev
```
